3.- Explique qué son las variables en JavaScript y cuál es la diferencia entre var, let y const. Incluya un ejemplo sencillo de cada una e indique cuál de estas formas ya no se recomienda utilizar actualmente y por qué.

R.- Las variables en JS guardan informacion o datos:

VAR: es para crear variables y se puede cambiar su valor pero puede dar problemas porque permite redeclarar variables
Por ejemplo:
var nombre = "Vladimir";
console.log(nombre);
nombre = "Harry";
console.log(nombre);
CONSOLA:
Vladimir
Harry

LET: tambien es para crear variables pero este es mas seguro y permite cambiar variables
Por ejemplo:
let edad = 22;
console.log(edad);
edad = 23;
console.log(edad);
CONSOLA:
22
23

CONST: Se usa cuando los valores no deben cambiar y no permite modificar el valor
Por ejemplo:
const pais = "Bolivia";
console.log(pais);
pais = "Argentina";
CONSOLA:
Mostrara error porque una constante no se puede modificar
